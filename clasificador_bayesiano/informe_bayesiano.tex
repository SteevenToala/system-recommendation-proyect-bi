\section{Objetivos}
\subsection*{Objetivo General}
Diseñar e implementar un clasificador bayesiano robusto para la prediccion de ingresos altos en el negocio de alquileres.
\subsection*{Objetivos Espec\'ificos}
\begin{enumerate}[label=\arabic*.]
  \item Aplicar la regla de decision optima $\psi_{opt}$ basada en probabilidades conjuntas.
  \item Calcular el error out-of-sample $E_{out}$ para validar la precision del modelo.
  \item Documentar el proceso matematico y estadistico de la clasificacion.
\end{enumerate}

\section{Introducci\'on}
El clasificador bayesiano es un pilar de la inteligencia de negocios, permitiendo transformar datos historicos en reglas de clasificacion probabilisticas.En este informe se detalla la aplicacion de este modelo sobre un dataset de alquileres de peliculas.\begin{cajainfo}{Variables del Modelo}Se utilizan $x_1$ (Popularidad) y $x_2$ (Valor del Genero) para predecir la variable $y$ (Ingreso Alto).\end{cajainfo}

\section{Fundamento Te\'orico}
Se emplea la regla de decision optima definida en clase para minimizar el error de clasificacion:
\begin{equation}
  \psi_{opt}(\mathbf{X}) = \begin{cases} 1, & \text{si } P(\mathbf{X}, y=1) \geq P(\mathbf{X}, y=0) \\ 0, & \text{Caso contrario} \end{cases}
\end{equation}
Asimismo, el error generalizado se calcula mediante la probabilidad de fallo sumada sobre todas las regiones de evidencia:
\begin{equation}
  E_{out}(\psi_{opt}) = \sum_{\{\mathbf{X}:\psi_{opt}(\mathbf{X})=1\}} P(y=0|\mathbf{X})P(\mathbf{X}) + \sum_{\{\mathbf{X}:\psi_{opt}(\mathbf{X})=0\}} P(y=1|\mathbf{X})P(\mathbf{X})
\end{equation}
