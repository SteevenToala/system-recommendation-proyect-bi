\section{Objetivos}
\subsection*{Objetivo General}
Implementar un clasificador bayesiano utilizando el dataset de alquileres para calcular probabilidades posteriores y tomar decisiones optimas de clasificacion.
\subsection*{Objetivos Espec\'ificos}
\begin{enumerate}[label=\arabic*.]
  \item Identificar caracteristicas binarias relevantes para el modelo.
  \item Calcular las frecuencias de ocurrencia para cada combinacion de variables.
  \item Determinar la clase optima basandose en la maxima probabilidad posterior.
  \item Evaluar el error total de clasificacion del modelo bayesiano.
\end{enumerate}

\section{Introducci\'on}
El teorema de Bayes es una herramienta fundamental en el aprendizaje supervisado.Permite calcular la probabilidad de que un evento pertenezca a una clase determinadadadas ciertas evidencias u observaciones. En este caso, analizamos el comportamientode los clientes de alquiler de peliculas para predecir si generan un ingreso alto.

\begin{cajainfo}{Variables Seleccionadas del Dataset}Para este estudio se han seleccionado las siguientes etiquetas binarias originales:\begin{itemize}  \item \textbf{x1:} \texttt{pelicula\_popular} (Indica alta demanda).  \item \textbf{x2:} \texttt{genero\_alto\_valor} (Indica generos premium).  \item \textbf{y:} \texttt{ingreso\_alto} (Variable objetivo a clasificar).\end{itemize}\end{cajainfo}

\section{Probabilidades Prioris}

\section{Conclusiones}
\begin{enumerate}
  \item El error total del modelo es de \textbf{0.3710}, lo cual indica que las variables actuales tienen limitaciones predictivas.
  \item La clase dominante en todas las combinaciones es la clase 0 (Ingreso Bajo).
  \item El teorema de Bayes permite cuantificar la incertidumbre en la toma de decisiones empresariales.
  \item Para mejorar la precision, se recomienda incluir mas caracteristicas o aumentar la granularidad de los datos.
\end{enumerate}
